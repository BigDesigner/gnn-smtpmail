# Contributing to GNN SMTPMail

Thank you for your interest in contributing to GNN SMTPMail!

## Bug Reports

If you find a bug, please create a new issue in the repository. Be sure to include:
-   Steps to reproduce the issue.
-   Your environment details (WordPress version, PHP version).
-   Any relevant error logs.

## Pull Requests

1.  Fork the repository.
2.  Create a new branch for your feature or fix (`git checkout -b feature/amazing-feature`).
3.  Commit your changes.
4.  Push to the branch.
5.  Open a Pull Request.

Please ensure your code follows WordPress Coding Standards.
